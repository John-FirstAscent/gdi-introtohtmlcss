Absolutely Positioning

An element positioned absolutely will ignore the normal flow of elements on a page. It will position itself relative to the first block element that has position other than static. If no element is found, the container block becomes html.

Overlapping Elements

Sometimes elements will overlap. You can change the stack order by adjusting the z-index. Elements with the highest z-index go on top.

CNN.com in Firefox's 3d Inspector

Overlapping Elements

CSS


						.pink {
						  background-color: pink;
						  position: absolute;
						  left: 10px;
						  top: 10px;
						  width: 100px;
						  height: 100px;
						}
						.blue {
						  background-color: blue;
						  position: absolute;
						  left: 20px;
						  top: 25px;
						  width: 100px;
						  height: 100px;
						}

Floating Elements

Floating Elements

Applying a float to an element essentially takes it out of the normal flow and "floats" it either as far left or as far right as its containing element will allow.

It should also be noted that other elements, such as paragraphs or lists, will inherently wrap around a floating element.

Floating Elements

CSS


						.yellow {
						  background-color: yellow;
						  padding: 10px;
						  width: 100px;
						  float: left;
						}
						p {
						  font-size: 12px;
						}

Clearing Floats

You might want some elements to float, but other
elements to start a new line. You can use a clear to achieve this:


						clear: left;
						

						clear: right;
						

						clear: both;
