Positioning Elements

So far we have encountered block elements. They appear on the next line, like paragraphs.

There are also inline elements. They appear on the same line that they are written on.

Block vs Inline Elements

CSS divides HTML into these two elements.

You can imagine block-level elements always rendering on a new line.

Inline Elements: img, a, br, em, strong

Block Elements: p, h1, ul, li (and almost everything else)

What about adding hover
states to links?

Pseudo-Classes

Pseudo-classes are a great way to target different "states" of a link or element. They are a special type of selector.

For example, the <a href="link.html"> element has 4 "states":

:active

:focus

:hover

:visited

Pseudo-Classes

CSS


						a:active {
						  color:pink;
						}
						

						a:focus {
						  color:blue;
						

						a:hover {
						  color:green;
						}
						

						a:visited {
						  color:purple;
						}

State

When clicking on a link, the button is :active when the button is depressed.

Using the tab key to go through links on a page will :focus the link (and adding a blue glow by default)

Using your mouse to hover over a link will trigger the :hover state of a link.

Clicking a link, then navigating back to the page with the original link will display the link as :visited

Adding Borders Around Elements

The border style sits between the padding and
margin and also adds to the size of the element.


						padding: 10px 20px 5px 7px;
						margin: 3px 3px 5px 3px;
						border: 1px solid red;

This element is rendered as 127 x 65px

Adding Borders

Border styles can be written by listing out each attribute:


						border-color: pink;
						border-style: dotted;
						border-width: 2px;

However, the industry standard is to use the shorthand declaration:

(size, style, style)


						border: 2px dotted pink;

Adding Borders

You can also declare specifically where a border should be:


						border-left: 2px solid green;
						border-bottom: 10px dotted #ff00ff;

Using Wrappers

Using classes called .wrappers are commonly found wrapping
around large groups of content. They are usually used for aligning the entirity of a page on a screen.


						div.wrapper {
						  width: 100%;
						  max-width: 1400px;
						  margin: 0 auto;
						}

Declaring Width

Sometimes you will want to declare the width of an element manually. Widths can be declared in percentages or pixels.


						div#sidebar {
						  width: 31%;
						  padding: 1%;
						  border: none;
						}

⚠️ Remember: A browser reads the height and width of an element
plus padding and border.
